Auth API Views
========================
- Auth API views file  **@ auth_/api/views.py**
- In this view we mainly used **Django REST framework** , feel free to check the: `Documentaion <https://www.django-rest-framework.org/>`_.


- Auth API View Contains *Three main Functions:
    
    - UserList
    - UserDetail
    - UpdateUser

**UserList** 
------------
- *GET Functions* :
    - **get** : Retrieves all CustomUsers

**UserDetail** 
--------------
- *GET Functions* :
    - **get** : Retrieves a CustomUser

**UpdateUser** 
--------------
Updates or deletes a CustomUser.

- **update** : update a CustomeUser
- **delete** : deletes a CustomeUser
