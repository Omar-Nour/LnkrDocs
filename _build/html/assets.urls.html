Assets URLS

  • Assets urls file @ assests/api/urls.py

  • URL Path : “api/assests/”

  • Assests urls file has 7 main urls representing all the APIs needed for Assets
    • gender/

      • url for getting genders that uses the API GenderViewSet in views.py file

    • week_day/

      • url for getting all week days that uses the API WeekDayViewSet in views.py file

    • insurance/

      • url for getting all insurance companys that uses the API InsuranceCoViewSet in views.py file

    • clinic_speciaity/

      • url for getting all clinic specialities that uses the API ClinicSpecialityViewSett in views.py file

    • physician_specialities/

      • url for getting all physician specialities that uses the API PhysicianSpecialitiesViewSet in views.py file

    • patient_diagnostic/

      • url for getting all patient diagnostics hat uses the API PatientDiagnosticsViewSet in views.py file

    • city/

      • url for getting all cities that uses the API CityViewSet in views.py file
